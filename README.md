# Магазин товаров (Shop)

## О проекте

Данный проект реализует объектно-ориентированную модель интернет-магазина с категориями товаров, продуктами и заказами.  
В проекте представлены следующие основные сущности:
- **Product** — базовый класс товара с поддержкой логирования, контроля изменения цены, а также возможностью создавать и обновлять товары.
- **Category** — класс для группировки товаров по категориям, поддерживает итерацию по товарам и статистику по количеству товаров и категорий.
- **Smartphone** и **LawnGrass** — специализированные классы товаров-наследников.
- **LoggerMixin** — миксин для автоматического логирования создания объектов.

## Основные возможности

- Добавление и вывод информации о товарах и категориях.
- Безопасное изменение цены с подтверждением снижения.
- Итерация по товарам в категории.
- Поддержка различных видов товаров через наследование.
