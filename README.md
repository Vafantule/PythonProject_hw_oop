# Product & Category Classes

## Описание

Проект содержит простые Python-классы `Product` и `Category` для моделирования товаров и их категорий.

## Структура

- `Product` — класс для хранения информации о товаре (название, описание, цена, количество).
- `Category` — класс для хранения информации о категории товаров и списка товаров этой категории, 
а также подсчёта общего количества категорий и товаров.

## Тестирование

Для тестирования используется библиотека `pytest`.  
Тесты расположены в файле `test_product_category.py`.

## Запуск тестов

```bash
pytest test_product_category.py
```
